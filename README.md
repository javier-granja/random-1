# random
Ejemplo del Módulo 5

1.- Hacer fork del proyecto https://github.com/jquemada/random GITHUB.
2.- clonamos el repositorio en nuestro ordenador local 
    git clone  https://github.com/javier-granja/random-1.git
    nos pone el origin del local en el url de GITHUB al reves pone pegas.
3 .-Añadimos los archivos con add de git
    git add READMR.md
    git add LICENSE.md
    git add .
4.- Conjelamos la versión
    git commit
5.- sincronizamos con la versión de GITHUB
    git push -u origin master
